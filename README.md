# Nexora Dashboard

Modernized Nexora Dashboard UI with responsive design, dynamic KPI tracking, and smooth alert expansion transitions.

## Features
- **Modern UI**: Clean, geometric design using Plus Jakarta Sans.
- **Dynamic KPIs**: Real-time status indicators for table issues and acceptance.
- **Smooth Transitions**: FLIP-powered animations for expanding alerts into detailed analysis views.
- **Responsive Sidebar**: Categorized navigation for easy access to monitoring and insights.

## Deployment on Render (Free Tier)
To host this dashboard on Render for free:
1. Log in to [Render](https://render.com/).
2. Click **New +** and select **Static Site**.
3. Connect your GitHub account and select the `Nexora_updated` repository.
4. Render will automatically detect the `index.html` and deploy it.
5. Your dashboard will be live on a `*.onrender.com` URL!

## Project Structure
- `index.html`: Main dashboard entry point.
- `styles.css`: Custom modern styling and layout logic.
- `script.js`: Interactive elements and smooth expansion animations.
- `procdna-logo.png`: Brand asset.
